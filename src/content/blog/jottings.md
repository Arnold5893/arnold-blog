已记不清从何时起，我似乎（主动/被动）放弃了对周围世界和未知事物的探索，取而代之的是那个只会整天琢磨怎么考高分、拿名次的机器。也许是缺少这方面的天赋，又或许只是我生性死板

单拿写作这方面来说，似乎从中学开始，我就很少在作文中吐露自己的真实想法，只会拙劣地模仿，而偶尔吐露出的真情实感，大多会被老师评价说是“跑题”、“流水账”。为了分数，我学会了妥协，开始套用甚至抄袭所谓的优秀范文。我知道这不是长久之计，也只是心安理得地得过且过（目光比较短浅）。到了高中，写作基本上变成了体力活，背几套放之四海皆准的模板，记下几个干瘪的人物事迹，等到考试的时候就想方设法把它套进去，就这样，像流水线的工人。这种方法也使得我高中三年的写作几乎没什么的进步，无非就是哪次的事例找的接近题目要求分数就高一些，反之就低一些

把视线拉长到整个学生生涯，我应该是更加的死板了。发散思维匮乏，知识整合能力太差，加上性格的“胆小”，这些缺点在高中三年暴露无遗，对很多学科的学习仅是停留在照本宣科，可我并不自知。那时的我，每天精打细算着额外的学习时长，把吃饭和睡觉的时间挤出来，只为整理几道下次做依然会做错的题，多背几个最后到考场还是没记住/没用上的知识点，似乎比别人多熬几分钟就能减缓我的焦虑。我们班甚至陷入了一种怪圈：“我已经学累了，瞥一眼旁边的同学还在做题，不行，他/她都那么厉害了，还在努力，我不能休息”，这种近乎自虐的盲目攀比，给我留下了一堆坏毛病——both physical and mental

曾有同桌说我太“教条”，我当时还暗想：“等我考上大学，你只能去专科，有什么好说我的“，于是不再理睬她（当然这里不是说两个人的关系不好，只是不再讨论这方面的问题）继续做题。可现在我却不得不承认，她一语中的，我确实不会变通。还记得有同学打趣我是“两耳不闻窗外事，一心只读圣贤书”——大概是因为我极少参与她们关于学校/娱乐圈的八卦或小说的闲聊。彼时的我还把这当作夸奖暗自窃喜，现在看来，这不过是自我封闭的另一种绝佳写照罢了

本以为到了大学，一切都会向好的方向发展，至少是会比高中好不少的，实则不然。大一上，我在短暂尝试了几个新方向后，又迅速缩回了高中那种循规蹈矩的壳里。宿舍、教室、图书馆、食堂，标准的四点一线，早出晚归。课上听天书，课下就扎进图书馆，对着 B 站的视频疯狂补习。记笔记、做题，一套动作行云流水，一如高中。虽然最终成绩还算能看得过去，但我自己是知道的：那些知识全是虚胖，一到需要实操环节就会原形毕露。印象最深的是，一个学期的 Python 课结束后，我连一个递归题都写不出来，真挺招笑的

“一向好强”的我自然不甘于此，但其实那个“胆小鬼”从来没变过。做什么都是心血来潮，然后慢慢的等到热情一退便草草收场。大一期间，我跟风看过吴恩达老师的《机器学习》，也刷过当时 B 站很火的数据分析网课。遇到听不懂的地方，第一反应依然是一直以来的思维惯性：找老师，换个网课继续看。我曾迷信“只要肯努力就一定会有收获”，这句话不错，但需要真努力，而不是用时长来麻痹自己的fake effort。现在想来，我一直对自己很吝啬，觉得把时间花在教材之外的探索上都是浪费，但实际上，在那些恪守本分的日常里，我也只是在得过且过

大二上的一次偶然探索，我摸索到了 GitHub 上备受推崇的csdiy（当时我连 GitHub 用不明白），顺着 CSDIY，我遇到了真正激起我学习兴趣的 CS50。在此之前，我的学习永远是按部就班，学校开什么课我学什么，书上印什么我记什么。虽然没有看完整套课程，但 David 教授、助教团队以及台下学生们对cs的那份纯粹的热爱，深深击中了我。对于一个上大学前几乎没怎么碰过电脑的人来说，那是第一次意识到，原来 CS 是如此鲜活有趣的，它远不止是课本上那些枯燥死板的符号

再后来，我开始着手啃 CSAPP，尝试用不同的 AI 做一些简单的小项目、小工具，尝试主动探索，让自己变得更 curious 些，尽管这些尝试对大部分 CSer 来说只是家常便饭，但我却很珍惜

最后，用一句我喜欢的人物的话来结尾：

  如果不是自己期望的未来就没有意义，  
  自己的价值能够由自己来找出，  
  只要明白这一点，那就够了了
